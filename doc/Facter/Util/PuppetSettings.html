    with_puppet() { || ... } click to toggle source

    This method is intended to provide a convenient way to evaluate a Facter code block only if Puppet is loaded. This is to account for the situation where the fact happens to be in the load path, but Puppet is not loaded for whatever reason. Perhaps the user is simply running facter without the â€“puppet flag and they happen to be working in a lib directory of a module.

def self.with_puppet
  begin
    Module.const_get("Puppet")
  rescue NameError
    nil
  else
    yield
  end
end
