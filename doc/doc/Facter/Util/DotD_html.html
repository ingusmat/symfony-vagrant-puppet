    <h1 class="class">class Facter::Util::DotD</h1>

<div id="description" class="description">

    <p>A <a href=“../../Facter.html”>Facter</a> plugin that loads facts from /etc/facter/facts.d and /etc/puppetlabs/facter/facts.d.</p>

    <p>Facts can be in the form of JSON, YAML or Text files and any executable that returns key=value pairs.</p>

    <p>In the case of scripts you can also create a file that contains a cache TTL. For foo.sh store the ttl as just a number in foo.sh.ttl</p>

    <p>The cache is stored in /tmp/facts_cache.yaml as a mode 600 file and will have the end result of not calling your fact scripts more often than is needed</p>

    </div><!-- description -->
